# ❤️ Heart Attack Risk Prediction System

## 📌 Project Overview
This is a Machine Learning based web application that predicts the risk of heart attack using patient health parameters.

The system takes user input through a web form and provides prediction results using a trained ML model.

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- HTML & CSS

---

## 📊 Machine Learning Model
- Data preprocessing performed
- Feature selection applied
- Model trained using classification algorithm
- Accuracy evaluated
